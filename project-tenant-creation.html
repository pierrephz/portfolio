Subscribing and running a network server.

When subscribing to a plan in our company, you need to create a tenant for your network. This is a feature that requires some UX research and some UI exploration.

User experience.

Wireframing and do research to defining the best UX for this use case. In this case, it will be the wizard pattern.

User interface.

Mocking up on Figma using a design system. Create a desktop and mobile version. If the use case is complex; create a prototype. Hand over to developers and organize feedback sessions.

Release the feature.

We first release our features on a preview server and make some user tests. Once everyone is satisfied, we can finally release online.
