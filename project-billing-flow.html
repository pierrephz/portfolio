SaaS company.

As a SaaS company, we need an adaptive billing process, with multiple prices plans. Must be user friendly, use our design system, follow UX rules.

User experience research.

First, clearly define the needs, align with the sales team on the different pricing plans (Discovery, Standard, Pro), what kind of payment method, create an account before check out, etc. Create some flows and different user scenarios. Create wireframes.

Mocking up on Figma and prototyping.

Shape this project, create a user friendly interface, make use of the design system. For a complex project like this, a prototype with multiple user flows is worth it for the developpers.

Review sessions and end to end testing.

Hand over the project to the developers, going throught feedback sessions. Then make some end to end tests with users. Make more changes according to potential feedbacks.
